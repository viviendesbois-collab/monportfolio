Construire ce portfolio avec Claude Code

20 juin 2026

Ce portfolio est volontairement minimaliste : du HTML sémantique, du CSS organisé en BEM et un peu de JavaScript vanilla. Pas de framework, pas de bundler. L'objectif était d'apprendre les fondamentaux du web tout en me faisant assister par l'IA pour aller plus vite.

Pourquoi rester sur du vanilla ?

En formation, comprendre ce qui se passe « sous le capot » compte plus que la rapidité de mise en place. Travailler sans abstraction m'oblige à maîtriser le DOM, la cascade CSS et les variables, ou encore l'API IntersectionObserver pour les animations au défilement.

Le rôle de Claude Code

Claude Code a servi de binôme : proposer une structure, expliquer chaque choix avant de l'appliquer, et garder une cohérence (palette, espacements, conventions de nommage) au fil des sections. Quelques exemples concrets :

  • une section Hero centrée et un système de cartes de projet réutilisable ;
  • un mode clair/sombre piloté uniquement par des variables CSS ;
  • un refactor pour extraire les valeurs « magiques » en tokens.

Et la suite ?

Ce blog en est la première brique : la liste est générée depuis un simple fichier JSON, mais chaque article reste une vraie page HTML. Prochaine étape : brancher les vrais liens et publier d'autres retours d'expérience.
